## What is this?

**VibeGam.ing** is a community-driven gallery for games created with the help of AI — prototypes, jams, weird experiments, and polished releases.

## Add your game in 60 seconds

1. Fork this repo
2. Edit `src/content/games.ts`
3. Open a Pull Request
4. After review + merge, your game goes live on vibegam.ing

## Local dev

```bash
pnpm i
pnpm dev
```

## Tech

- Next.js (App Router) + TypeScript
- Tailwind CSS
- next-intl (i18n)
- Static content from `src/content/games.ts`
